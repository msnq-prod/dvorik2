# API Documentation

This document describes the API for the Telegram bot and admin panel application.

*This documentation is currently a work in progress.*

## Base URL

`/api`

## Endpoints

### Users

*   `POST /users/` - Create a new user.
*   `GET /users/` - Get a list of users.
*   `GET /users/{user_id}` - Get a specific user.

### Coupons

*   `POST /coupons/` - Create a new coupon.
*   `GET /coupons/` - Get a list of coupons.
*   `GET /coupons/{coupon_id}` - Get a specific coupon.

### Campaigns

*   `POST /campaigns/` - Create a new campaign.
*   `GET /campaigns/` - Get a list of campaigns.
*   `GET /campaigns/{campaign_id}` - Get a specific campaign.
